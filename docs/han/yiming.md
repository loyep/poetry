# 佚名

## 饮马长城窟行

版本一：

青青**河畔**草，绵绵思远道。  
远道不可思，宿昔梦见之。  
梦见在我傍，忽觉在他乡。  
他乡各异县，展转不相见。  
枯桑知天风，海水知天寒。  
入门各自媚，谁肯相为言？  
客从远方来，遗我双鲤鱼。  
呼儿烹鲤鱼，中有尺素书。  
长跪读素书，**其中意**何如？  
上言加**餐饭**，下言长相忆。
